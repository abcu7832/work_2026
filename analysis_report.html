260518_tray4 PASS 분석 리포트

핵심 요약

이 리포트는 `V_rx`와 `bias` 조건별 결과를 `All pass(50/50)`와 `Not all pass` 2개 클래스로 나누어 봅니다. `pass_rate`는 `PASS 횟수 / 50`이며, 예를 들어 `49/50`은 `pass_rate=0.98`입니다.

  • 분석 대상은 CSV 55개, 조건 3080개입니다.
  • V_rx는 700-1300 mV, bias는 0-7 범위입니다.
  • 전 파일/전 bias 공통 All pass(50/50) 구간: ng(불량): 700 mV
  • PASS 영역 개요는 ok/ng별 subplot 2개로 묶고, 전체 파일 기준 최저 PASS 횟수와 50/50 이진 분류를 같은 figure 안에서 비교합니다.
  • PASS 연속 구간 클래스는 상하좌우로 연결된 All pass/Not all pass 영역을 하나의 region으로 묶은 결과입니다.
  • 파일별 Shmoo grid는 양품/불량과 All pass/Not all pass 클래스를 선택하여 파일 간 패턴 차이를 비교하도록 구성했습니다.

데이터 개요

입력 CSV의 조건 수, `V_rx`, `bias`, `V_core`, 클래스 분포를 먼저 확인합니다. `V_core`가 단일 값이면 이번 리포트에서는 판정 변수로 쓰지 않습니다.

PASS 영역 개요

가로축은 `bias`, 세로축은 `V_rx`입니다. 각 샘플 구분별로 레벨 Shmoo와 이진 Shmoo를 하나의 subplot figure 안에 좌우로 배치해 동일한 크기로 비교합니다.

ok(양품)

ng(불량)

마지막 Shmoo는 ok와 ng 전체 파일을 모두 포함해 같은 이진 기준으로 본 결과입니다. 전체 샘플을 동시에 만족하는 조건을 확인할 때 사용합니다.

파일별 Shmoo grid

콤보박스로 양품/불량 및 표시 클래스를 선택합니다. 선택된 클래스의 모든 파일을 grid로 배치하여 파일 간 패턴 차이를 한 화면에서 비교합니다.

PASS 연속 구간 클래스

전체 파일 기준 최저 PASS 횟수로 `All pass(50/50)`와 `Not all pass`를 먼저 나눈 뒤, `V_rx-bias` 평면에서 상하좌우로 연결된 영역을 하나의 클래스(region)로 묶었습니다. 예를 들어 `V_rx 1100-1300 mV, bias 0-2`가 연결되어 모두 All pass이면 하나의 영역 클래스로 표시됩니다.
